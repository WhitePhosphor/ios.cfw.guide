Using Cydia

如果需要英语帮助，可以到 r/Jailbreak Discord Server 上进行询问。

# 读前须知

TIP

如果你对越狱不是很熟悉，那么它值得一看。如果很熟悉的话你可以选择不看这个。

Cydia 是一个自 2008 年越狱兴起以来就被使用的包管理器，它可以用来安装插件和系统主题。

插件（Tweaks） 是对于系统的一些小的修改，可以改变系统的外观和使用体验。你可以在一些仓库中找到插件，通常这些仓库被称为 "repos"。

仓库（Repo） 本质上是包含大量插件的数据库。当你将仓库添加到 Cydia 时，你就可以安装其中的插件，并使用搜索功能查找它们。

本指南将向你展示如何安装和卸载插件，以及如何添加不同的仓库。

# 添加仓库

  1. 打开 Cydia 应用程序
  2. 点击底部栏上的 Sources
  3. 点击右上角的 Edit 按钮
  4. 点击右上角的 + 图标
  5. 输入仓库 URL（例如 https://sparkdev.me/在新窗口中打开）
  6. 点击 Add Source

现在你可以从此页面进入仓库，查看其中所有可用的插件。

# 删除仓库

  1. 打开 Cydia 应用程序
  2. 点击底部栏上的 Sources
  3. 点击左上角的 Edit 按钮
  4. 点击你想删除的仓库旁边的红色图标
  5. 确认删除

现在这个仓库应该已从 Cydia 中删除。

# 安装插件

  1. 打开 Cydia 应用程序
  2. 搜索你想安装的插件
  3. 点击屏幕右上角的 Install 按钮
  4. 确认安装
    • 安装插件可能需要一些时间，这取决于你的网速和插件的大小
  5. 完成后，点击 "重启 SpringBoard"
    • 设备在此之后应该会刷新（Respring）
    • 有时，"重启 SpringBoard"可能不会显示，但你仍然可以点击按钮

该插件现在应该已安装到你的设备上，并将在设备越狱时激活。

# 卸载插件

  1. 打开 Cydia 应用程序
  2. 点击你想卸载的插件
  3. 点击页面顶部的 Modify 按钮
  4. 点击 Remove
  5. 确认卸载
    • 卸载插件可能需要一些时间
  6. 完成后，点击 "重启 SpringBoard"
    • 设备在此之后应该会刷新
    • 有时，"重启 SpringBoard"可能不会显示，但你仍然可以点击按钮

重启 SpringBoard 后，该插件现在应该已从你的设备上卸载。
